1 Abstract

The Smart Energy Consumption Meter is an intelligent system designed to monitor, analyze, and optimize household electricity usage in real time. The system measures voltage, current, power (W), energy consumption (kWh), and power factor using sensors and a microcontroller-based processing unit.


The collected data is displayed on an LCD/OLED screen and transmitted to a cloud-based IoT platform for remote monitoring. The system also includes smart load control using relay modules.

2 Introduction

With increasing electricity demand and rising energy costs, efficient energy monitoring has become essential. Traditional energy meters only measure total consumption without providing real-time insights.


The Smart Energy Consumption Meter provides real-time monitoring, data logging, and intelligent load management to help users track and reduce energy usage.

3 Problem Statement

  • Lack of real-time electricity monitoring in homes
  • No visibility of individual appliance energy usage
  • Energy wastage due to inefficient usage
  • No predictive analysis for electricity consumption

4 Objectives

  • To design a real-time energy monitoring system
  • To measure voltage, current, power, and energy consumption
  • To display live electrical parameters
  • To transmit data to cloud for remote access
  • To enable smart load control for energy saving

5 System Components

  • AC Mains Supply (230V)
  • Current Transformer (CT Sensor)
  • Voltage Sensing Module
  • ESP32 / Arduino Microcontroller
  • LCD / OLED Display
  • Relay Module (Smart Load Control)
  • WiFi Module (Built-in ESP32)
  • PCB Board
  • IoT Cloud Platform

6 System Architecture

AC Supply→CT + Voltage Sensor→ESP32
ESP32→LCD Display
ESP32→WiFi→IoT Cloud
ESP32→Relay→Loads
System Schematic Diagram

7 Working Principle

  • The AC mains supply powers household appliances
  • The CT sensor measures current flowing through the live wire
  • The voltage sensor measures supply voltage
  • The microcontroller calculates Voltage, Current, Power, Energy, and Power Factor
  • Values displayed on LCD
  • Data sent to IoT cloud via WiFi
  • Relay module can disconnect loads during overload

8 Mathematical Calculations

// Power Calculation
Power (W) = Voltage (V) × Current (A)

// Energy Calculation
Energy (kWh) = Power (W) × Time (hours) / 1000

// Power Factor
Power Factor = Real Power / Apparent Power
